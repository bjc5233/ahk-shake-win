# ahk-shake-win
> 当输入指定字符时抖动当前窗口


## 使用
1. 安装autoHotKey软件
1. 修改shakeWin.ahk中a按键为自己想要的按键；修改offset变量为抖动幅度；修改loopNum变量为抖动次数
1. 双击shakeWin.ahk执行
1. 在任意可输入的地方（如记事本）输入[ a ]，当前窗口会抖动指定次数
